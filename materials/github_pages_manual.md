# Webサイト（紹介用URL）のGitHub Pages公開手順

LINE公式アカウントのプロフィールなどに設定する「紹介用URL」として、自前のWebサイトを無料で公開する手順です。

## 準備するもの
1.  **紹介用ページ（HTMLファイル）**
    *   事前に用意した `index.html`（「古民家カフェ 結」案内ページ）をお手元のパソコンに保存しておきます。
2.  **GitHubアカウント**
    *   まだお持ちでない場合は、[GitHub公式サイト](https://github.com/)（右上の「Sign up」）から無料で作成してください。

---

## 手順1：GitHubに新しいリポジトリ（置き場所）を作る
1.  GitHubにログインし、画面右上の「＋」アイコンから **New repository** をクリックします。
2.  **Repository name**（リポジトリ名）を入力します。
    *   例：`cafe-yui-demo` や `my-cafe` など、半角英数字で好きな名前をつけます。
3.  **Public**（公開）が選択されていることを確認します（※Pages機能を使うため必須です）。
4.  一番下の緑のボタン **Create repository** をクリックします。

---

## 手順2：HTMLファイルをアップロードする
1.  リポジトリ作成直後の画面で、**uploading an existing file** という青いリンクをクリックします。（または「Add file」>「Upload files」を選択）
2.  「Drag files here to add them to your repository」と書かれた枠の中に、準備しておいた `index.html` ファイルを **ドラッグ＆ドロップ** します。
3.  下部の「Commit changes」ボタン（緑色）をクリックして、アップロードを完了させます。

---

## 手順3：GitHub Pages（Web公開機能）を有効にする
1.  アップロードしたリポジトリの画面上部にあるタブから **Settings**（歯車マーク）をクリックします。
2.  左側のメニューを下の方にスクロールし、**Pages** をクリックします。
3.  「Build and deployment」という項目の下にある **Source** が `Deploy from a branch` になっていることを確認します。
4.  その下の **Branch** の部分で、`None` と書かれているドロップダウンを開き、`main`（または `master`）を選択します。
5.  隣のボタンが `/ (root)` になっていることを確認し、**Save** をクリックします。

---

## 手順4：URLが発行されるのを待つ（数分かかります）
1.  Saveを押すと、画面の上部に「Your site is ready to be published at `https://◯◯.github.io/リポジトリ名/`」というメッセージが表示されます。（すぐには表示されず、数分待ってからページを再読み込み（リロード）すると表示されることが多いです）
2.  そのURLが、**あなたの紹介用WebサイトのURL** になります。

---

## 手順5：LINE公式アカウントに設定しよう！
1.  発行されたURL（ `https://〜` ）をコピーします。
2.  LINE公式アカウントの管理画面（Web版）を開き、「プロフィール」設定画面へ行きます。
3.  プロフィールの「基本情報」や「ボタン（リンク）」の設定箇所に、コピーしたURLを貼り付けます。
4.  これで、LINEのプロフィールから先ほどのカフェのページに飛べるようになります！
